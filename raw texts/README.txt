The folder contains all extracted sections from all 21,520 xml files.
Each line in the txt corresponds to one file.
The name of the file and the extracted section are tab-separated.
The first column corresponds to the name of the file and the second column to the 
extracted text/section.

The introductions.txt file is accessible from here: https://drive.google.com/file/d/1rFl-5DEDkd6vpcH2Hf0svCpIELCJKpAy/view?usp=sharing. Since its size exceeds 25MB, it could not be uploaded to the repository.
